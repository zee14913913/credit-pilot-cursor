# ⌨️ 键盘操作粘贴 Token（不用鼠标）

## 🎯 不用鼠标的方法

### 方法1：使用菜单栏快捷键（推荐）⭐

**当提示输入 Password 时：**

1. **按 `Control + F2`**（聚焦到菜单栏）
2. **按方向键选择 "编辑"（Edit）**
3. **按回车**
4. **按方向键选择 "粘贴"（Paste）**
5. **按回车**

**或者更简单：**

1. **按 `Command + V`**（直接粘贴）
   - 即使有钥匙图标，`Command + V` 通常也能工作
   - 试试看！

---

### 方法2：使用终端快捷键

**当提示输入 Password 时：**

1. **按 `Command + V`**（尝试直接粘贴）
2. **如果不行，按 `Shift + Command + V`**
3. **或者按 `Control + V`**

---

### 方法3：手动输入（虽然看不到）

如果快捷键都不工作：

1. **从 GitHub 页面复制 Token**
2. **在终端中手动输入**（虽然看不到字符，但输入是正确的）
3. **输入完成后按回车**

**提示：** 即使看不到字符，输入是正确的，直接按回车就行！

---

### 方法4：使用触控板手势

如果你有触控板：

1. **当提示输入 Password 时**
2. **用两个手指点击触控板**（相当于右键）
3. **选择 "粘贴"**

---

## 🎯 推荐操作流程

```
1. 打开终端（Command + 空格，输入"终端"）
   ↓
2. 执行命令：
   cd /Users/1491-3913zee/Projects/CreditPilot
   git push -u origin main
   ↓
3. 提示 Username: 输入 1491-3913zee，按回车
   ↓
4. 提示 Password（有钥匙图标）:
   - 先试试：Command + V
   - 如果不行：手动输入Token（虽然看不到，但输入是正确的）
   - 输入完成后按回车
   ↓
5. 完成！✅
```

## 💡 重要提示

### 为什么看不到字符？

**这是完全正常的！** 为了安全，终端在输入密码时不会显示任何字符。

**即使你看不到，输入是正确的，直接按回车就行！**

### 如何确认输入成功？

- **方法1：** 直接按回车，如果推送成功，说明输入正确
- **方法2：** 输入后，再按一次 `Command + V`（确保粘贴成功）

---

## ✅ 最简单的方法

**直接按 `Command + V` 试试！**

即使有钥匙图标，`Command + V` 通常也能工作。

如果不行，就手动输入 Token（虽然看不到，但输入是正确的）。
