## 🔑 关键配置

### DATABASE_URL 格式
```
${{ Postgres.DATABASE_URL }}
```
⚠️ 双大括号，有空格

### Cron Schedule
```
0 14 * * *
```
每天 UTC 14:00（马来西亚时间 22:00）

### Start Command
- **Web 服务**：留空
- **Reminder-Cron 服务**：`cd backend && python3 reminder_system.py`

### 端口
```
8000
```

### 路径
- 上传目录：`/app/uploads`
- 报告目录：`/app/reports`

## ⚠️ 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 数据库连接失败 | DATABASE_URL 格式错误 | 使用 `${{ Postgres.DATABASE_URL }}` |
| 端口错误 | 端口配置不一致 | 确认所有地方都是 `8000` |
| 路径错误 | Volume 未挂载 | 创建并挂载 Volume |
| Cron 不执行 | Cron Schedule 格式错误 | 使用 `0 14 * * *` |
| 邮件发送失败 | 环境变量未设置 | 设置所有邮件相关变量 |
