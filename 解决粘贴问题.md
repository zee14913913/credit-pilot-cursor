# 🔓 解决 Token 粘贴问题

## 🔍 问题：有钥匙图标，无法粘贴

这可能是 macOS 的安全设置或钥匙串访问在拦截粘贴操作。

---

## ✅ 解决方法

### 方法1：使用右键菜单粘贴（最简单）⭐

1. **当提示输入 Password 时**
2. **在终端中右键点击**（鼠标右键）
3. **选择 "粘贴"** 或 "Paste"
4. **Token 就会粘贴进去**
5. **按回车**

---

### 方法2：使用菜单栏粘贴

1. **当提示输入 Password 时**
2. **点击终端菜单栏的 "编辑"**（Edit）
3. **选择 "粘贴"**（Paste）
4. **Token 就会粘贴进去**
5. **按回车**

---

### 方法3：手动输入（虽然看不到）

如果粘贴还是不工作，可以：

1. **从 GitHub 页面复制 Token**
2. **在终端中手动输入**（虽然看不到字符，但输入是正确的）
3. **输入完成后按回车**

---

### 方法4：临时禁用钥匙串访问

1. **打开 "系统设置"**（System Settings）
2. **搜索 "钥匙串"** 或 "Keychain"
3. **暂时关闭相关安全设置**
4. **推送完成后可以再打开**

（但这个方法比较复杂，不推荐）

---

## 🎯 推荐方法

**推荐使用方法1（右键菜单粘贴）**，最简单可靠！
