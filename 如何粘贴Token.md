# 📋 如何粘贴 Token - 详细步骤

## 🎯 在哪里粘贴 Token？

**在终端（Terminal）中粘贴！**

---

## 📋 完整步骤（一步一步）

### 第一步：打开终端

1. **按 `Command + 空格`**（打开 Spotlight）
2. **输入 "终端" 或 "Terminal"**
3. **按回车**（打开终端应用）

### 第二步：进入项目目录

**在终端中输入（复制粘贴）：**

```bash
cd /Users/1491-3913zee/Projects/CreditPilot
```

**按回车**

### 第三步：执行推送命令

**在终端中输入（复制粘贴）：**

```bash
git push -u origin main
```

**按回车**

### 第四步：粘贴 Token（重要！）

**执行命令后，终端会显示：**

```
Username for 'https://github.com':
```

**这时：**
1. **输入：** `1491-3913zee`
2. **按回车**

**然后终端会显示：**

```
Password for 'https://1491-3913zee@github.com':
```

**这时粘贴 Token：**

1. **在终端中点击鼠标**（确保光标在密码输入框）
2. **按 `Command + V`**（粘贴token）
   - ⚠️ **注意：** 虽然你看不到任何字符显示，但token已经粘贴进去了
   - 这是正常的！为了安全，密码输入时不会显示
3. **按回车**

---

## 💡 重要提示

### 为什么看不到字符？

**这是正常的！** 为了安全，终端在输入密码时不会显示任何字符（包括 `*` 或 `•`）。

**即使你看不到，token 已经粘贴进去了，直接按回车就行！**

### 如何确认粘贴成功？

- **方法1：** 直接按回车，如果推送成功，说明token正确
- **方法2：** 粘贴后，按 `Command + V` 再粘贴一次（确保粘贴成功）

## 🆘 如果粘贴不成功

### 方法1：手动输入

如果 `Command + V` 不工作，可以：
1. 打开token（在浏览器或笔记中）
2. 手动输入（虽然看不到，但输入是正确的）

### 方法2：使用右键菜单

1. 在终端中右键点击
2. 选择 "粘贴"

### 方法3：使用菜单

1. 在终端菜单栏点击 "编辑"
2. 选择 "粘贴"
