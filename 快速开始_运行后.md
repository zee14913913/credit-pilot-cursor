## 📋 常用操作

### 上传账单
```
POST /api/statements/upload
上传 Alliance Bank PDF 文件
```

### 查看账单列表
```
GET /api/statements
```

### 查看 Dashboard
```
GET /api/dashboard/stats
```

### 上传单据
```
POST /api/documents/upload
上传 Merchant Receipt / Payment Slip / Transfer Slip
```

### 下载 Excel 报告
```
GET /api/reminders/daily-report
```
